# Roadmap

## V1: Current MVP

- Private login and organization-aware app shell
- Companies, contacts, opportunities, pipeline, tasks, activities
- Document center and vendor document checklist
- Worker/freelancer database
- AI-assisted drafts and lead scoring route
- CSV import/export foundation
- Protected external lead import endpoint
- Supabase schema, RLS and private Storage bucket

## V2

- Gmail/Outlook sync for logged outreach
- Proposal generator
- PDF export for document templates and proposals
- Advanced worker availability
- Crew assignment planning
- Rate calculator
- Project delivery module
- Timesheets and daily reports
- Document approval workflow
- More detailed admin/user invitation UI

## V3

- Web scraping/import agents
- Public lead enrichment
- Tender monitoring
- Automated company research
- Data-center/MEP/EPC project monitor
- Public website integration
- Client portal
- External document sharing links with expiry

## Guardrails

- No automated LinkedIn scraping in the MVP.
- No automated mass emailing.
- Keep source URL and data provenance for every imported lead.
- Keep legal/compliance templates marked as drafts until reviewed by qualified advisor.
